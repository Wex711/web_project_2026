My Project

This semester I will be working on this web project. I have been given creative freedom regarding what kind of website I can create; what I plan to do is create a functional pokedex with integrated sounds and graphics. A pokedex is an encyclopedia from the pokemon franchise which stores information about all of the pokemon including their cries, appearance, size, etc. I want to have all of this information included in my project. It is also a goal of mine to make the website as accurate to the pokedex in the games as possible. I will be including the original 151 pokemon. I want to include a graphic for each pokemon. This is what a Pokedex looks like.

Future Endeavors

If I find it worthwhile, I plan to keep updating this project in the future with more and more pokemon over time. If I do, I will start by adding pokemon from newer generations to all pages, including all relevant information about the pokemon including data like height and weight. If I can, I want to make the website interactive by using videos and sounds if I can find a way to insert them. My main goal for this project is to make something that pokemon fans can enjoy and can navigate in their free time, which is why I will to to be as detailed as possible.
