About this Project

This semester, I have been assigned to create a web project of my choice. I have been given creative freedom regarding the topics of the website, and as such, I plan to create functional pokédex with sounds and graphics. A pokédex is an encyclopedia from the pokémon franchise which stores information about pokémon, including their cries, height, weight, etc. My goal is to make a virtual version of this device. I will be including the original 151 pokémon and their information.

Future Updates

If I find it worthwhile, I plan to keep adding pokémon from new releases over time. If I do so, I will go generation by generation of pokémon and add them in blocks. If I find a way to do so using the information from course material, I will also integrate pokémon cries by using sound. My goal for this project is to make a page pokémon fans can look through in their free time.
