Project Overview

I have been assigned to develop a web-based project of my choosing. Given the creative flexibility of the assignment, I have chosen to design a functional virtual Pokédex. A Pokédex is an in-universe encyclopedia from the Pokémon franchise that catalogs detailed information about various Pokémon species, including attributes such as their height, weight, abilities, and characteristics. The objective of this project is to recreate a digital version of this tool in the form of a fully interactive website.

Future Development

If the project proves to be manageable, I plan to expand the Pokédex beyond the original 151 Pokémon. Future updates will be implemented incrementally, adding new generations of Pokémon in structured phases. These expansions will continue to build upon the existing framework, incorporating additional features, improved functionality, and refined design elements as I apply new skills gained throughout time.
