Project Overview

This project is a virtual Pokédex. A Pokédex is an in-universe encyclopedia from the Pokémon franchise that catalogs detailed information about various Pokémon species, including attributes such as their height, weight, abilities, and characteristics. The objective of this project is to recreate a digital version of this tool in the form of an interactive website.

Future Development

I plan to expand the Pokédex beyond the original 151 Pokémon. Future updates will be implemented incrementally, adding new generations of Pokémon in structured phases. These expansions will continue to build upon the existing framework, incorporating additional features, improved functionality, and refined design elements.
