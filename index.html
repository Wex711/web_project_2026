Project details

This semester, I have been assigned to create a web project of my choice. My peers and I have been given creative freedom regarding the topics of our website, and as such, I have chosen to create functional pokédex with integrated sounds and graphics. A pokédex is an encyclopedia from the pokémon franchise which stores information about pokémon, including their cries, height, weight, etc. My goal is to make a virtual version of this device, with the intention of being as detailed as possible regarding pokémon descriptions. I want to make this website as accurate as possible to the pokédex seen in the pokémon/anime, and will be including the original 151 pokémon and their information.

Future updates

If I find it worthwhile, I plan to keep adding pokémon from new releases over time. If I do so, I will go generation by generation of pokémon and add them in blocks. If I find a way to do so using the information from course material, I will also integrate pokémon cries by using sound bytes. My goal for this project is to make a page pokémon fans can look through in their free time.
