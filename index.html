Future Updates

If I find it worthwhile, I plan to keep adding pokémon from new releases over time. If I do so, I will go generation by generation of pokémon and add them in blocks. If I find a way to do so using the information from course material, I will also integrate pokémon cries by using sound bytes. My goal for this project is to make a page pokémon fans can look through in their free time.
