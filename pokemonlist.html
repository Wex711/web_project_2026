Pokedex List Page

On this page I will keep an ordered list of the pokemon from the original 151. I intend to recreate the look of the pokedex from the games, which can be seen in this photo, by using tables and lists to organize it the way it appears in the image. If I can, I want to add a frame around the table so that it looks the way it does in the games.
